# Portfolio Website Design — Sena Flo

## Overview
Rebuild Sena's Wix portfolio as a polished Astro + Tailwind site for GitHub Pages. Same content structure, dramatically better design and performance.

## Pages
1. **Home** — Hero with FSAE car photo, portfolio intro, project gallery grid (3 projects)
2. **About** — Bio ("Hey There"), photo mosaic (guitar, fencing, travel, nature)
3. **Projects/Gallery** — Grid of project cards linking to detail pages
4. **Project Detail** (x3) — Rear Wing Aero, Steering Suspension, Sudoku — each with title, description, hero image, supporting images
5. **Resume** — Embedded PDF + download link

## Tech Stack
- Astro (static site generator)
- Tailwind CSS
- View Transitions API (smooth page navigation)
- CSS scroll-driven animations (subtle reveals)
- GitHub Actions for deployment

## Visual Direction
- Warm off-white/cream background (matching Wix feel but elevated)
- Clean sans-serif body (Inter), refined serif headings
- Generous whitespace, editorial feel
- Subtle hover effects on project cards
- Smooth page transitions
- Fully responsive

## Content
Matches existing Wix site exactly. Images from docs/ folder.
